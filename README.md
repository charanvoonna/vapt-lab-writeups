# VAPT Lab Writeups

This repository contains my hands-on **Vulnerability Assessment and Penetration Testing (VAPT)** practice reports.

The goal of this repository is to document my learning journey in **Offensive Security, Web Application Security, and Penetration Testing** through practical labs and exercises.

All reports are written in a **professional penetration testing format** including enumeration, vulnerability analysis, exploitation steps, impact assessment, and remediation.

---

# Objectives

The purpose of this repository is to:

- Practice real-world vulnerability exploitation
- Develop structured penetration testing documentation
- Build a public cybersecurity portfolio
- Demonstrate practical offensive security skills

---

# Report Structure

Each lab report follows a consistent pentesting report format:

Title
Overview
Enumeration
Vulnerability
Exploitation
Impact
Remediation
Screenshots


This structure helps simulate the format used in **professional penetration testing reports**.
